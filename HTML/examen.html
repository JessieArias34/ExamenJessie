Diferencias entre etiqueta "div" y etiqueta "p"

La etiqueta div es para dividir elementos en html, y la etiqueta p es para crear párrafos

¿Qué es minificación en Css?

Es hacer el archivo de css más pequeño, esto nos sirve para que la página web no sea pesada y usar solo los recursos necesarios y hacer más liviano nuestro trabajo

  • Quizá no en todos los navegadores se pueda abrir

¿Cómo poner variables en CSS?

Usamos :Root, ponemos dos guiones medios para definirla, despues el nombre y al final el valor que le vamos a asignar

¿Qué es un Pre-procesador en Css?

Por medio de JS se compila el css y así se pueden compliar los codigos y hacer el trabajo más rapido y ágil para nuestra página web. En si es reducir nuestro código.

Diferencia entre InnerHTML y Write:

Con write - agregas contenido directamente en el navegador. Con InnerHtml - cambias el contenido existente

¿Qué es una libreria?

En las librerias es un archivo o conjunto de archivos que te facilitan la programación. Existen ya componentes pre-frabricados para tu página que solo incluyes en tu código y así te facilita la construcción de la misma.

En JavaScript ¿qué es "getElementById"?

Seleccionar y llamar a un elemento del documento por medio del Id que le asignamos

Menciona 3 eventos para ejecutar una función en JavaScript:

  • Onclick
  • OnMouseover
  • onmouseout
